# Mai Manzil — Premium Event Experiences

A fully static, section-based website for Mai Manzil built with Next.js App Router and TailwindCSS.

## Setup

```powershell
npm install
```

## Development

```powershell
npm run dev
```

## Production build + static export

```powershell
npm run build
```

The static output will be generated in the `out` directory (Next.js `output: export`).

## Content editing

All site content is centralized in:

- `src/content/site.js`

Update text, links, services, portfolio items, and images there.

## Routes

- `/` Home
- `/portfolio`
- `/about`
- `/contact`

## Notes

- Placeholder images live in `public/images`.
- `robots.txt` and `sitemap.xml` are in `public/`.
